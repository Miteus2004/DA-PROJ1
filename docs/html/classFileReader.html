Detailed Description

A class responsible for reading data from files and setting up a graph for the Edmonds-Karp algorithm.

FileReader reads data from CSV files containing information about reservoirs, stations, cities, and pipes. It constructs a graph representing the network and sets it up for finding the maximum flow using the Edmonds-Karp algorithm.

void FileReader::Setup ( )

Set up the graph for the Edmonds-Karp algorithm.

Sets up the graph for finding the maximum flow using the Edmonds-Karp algorithm. Constructs main source and main target vertices, adds edges, calculates maximum flow, and writes the output to a CSV file. @complexity O(V * E^2) where V equals the number of vertexes and E the number of edges in the graph.
